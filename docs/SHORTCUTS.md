# Keyboard Shortcuts

Quick reference for Majoor Assets Manager shortcuts. For full details, see [HOTKEYS_SHORTCUTS.md](HOTKEYS_SHORTCUTS.md).

## Grid View

| Shortcut | Action |
|----------|--------|
| **1-5** | Set Rating |
| **0** | Clear Rating |
| **Enter / Space** | Open Viewer |
| **D** | Toggle Details Sidebar |
| **T** | Edit Tags |
| **B / Shift+B** | Add/Remove Collection |
| **Ctrl+A / Ctrl+D** | Select/Deselect All |
| **F2** | Rename |
| **Del** | Delete |
| **Ctrl+Shift+C** | Copy Path |
| **Ctrl+Shift+E** | Open in Explorer |
| **Ctrl+F / Ctrl+K** | Focus Search |

## Viewer

| Shortcut | Action |
|----------|--------|
| **Esc** | Close Viewer |
| **Space** | Play/Pause |
| ** / ** | Prev/Next Asset (or Step Frame if Player Focused) |
| **F** | Fullscreen |
| **D** | Info Panel |
| **I** | Pixel Probe |
| **L** | Loupe |
| **Z** | Zebra |
| **G** | Grid Overlay |
| **Alt+1** | 1:1 View |
| **+ / -** | Zoom |
