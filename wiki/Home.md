# Majoor Assets Manager Wiki

Majoor Assets Manager is a media library for ComfyUI outputs, with deep workflow inspection and reuse tools.

The simplest way to understand the extension is:

1. **Browse** your generated assets.
2. **Inspect** prompts, metadata, workflows, and visual details.
3. **Reuse** assets and workflow context directly in ComfyUI.

## Quick Start: 5 Things You Can Do

1. Open the Assets Manager sidebar tab and browse Outputs, Inputs, Custom roots, or Collections.
2. Search and filter by filename, prompt, metadata, media type, rating, workflow, date, size, or resolution.
3. Double-click an asset to inspect media, prompts, workflow metadata, and generation info.
4. Use the Majoor Floating Viewer for live generations, A/B comparison, side-by-side review, pins, and Graph Map workflow inspection.
5. Reuse assets by dragging to loader nodes or the canvas, staging to input, exporting clean files, or downloading selected files as a ZIP.

## Reuse

The same Assets Manager workflow covers reuse:

- drag assets to compatible loader nodes or the canvas
- stage files into ComfyUI input
- load selected media as nodes
- export originals, clean files, or selected files as a ZIP

See [Using the Assets Manager](Using-the-Assets-Manager) for the daily workflow.
